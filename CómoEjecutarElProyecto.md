# Instalación

## 1. Clonar el repositorio

```bash
git clone https://github.com/GeroRS33/PortalWEB.git
```

## 2. Backend

```bash
cd backend
npm install
npm start
```

## 3. Frontend

En otra terminal:

```bash
cd frontend
npm install
npm run dev
```

## 4. MongoDB Atlas

El proyecto utiliza una base de datos MongoDB Atlas ya configurada.

La IP pública del clúster permite conexiones desde cualquier dirección (0.0.0.0/0), por lo que no es necesario realizar ninguna configuración adicional.

## 5. Acceso

Abrir:

http://localhost:5173
